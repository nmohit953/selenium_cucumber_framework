Selenium Cucumber Framework (Java 21, TestNG, Cucumber, Selenium 4.26+)
=======================================================================

How to import:
1. Unzip selenium_cucumber_framework.zip
2. In Eclipse: File -> Import -> Existing Maven Projects -> select folder
3. Ensure your IDE uses JDK 21 or later (Project -> Properties -> Java Compiler)
4. Run tests: Right-click testng.xml -> Run As -> TestNG Suite

Notes:
- Selenium Manager (built into Selenium) will auto-download matching browser drivers
- Reports go to /reports, logs to /logs
- Test data located in src/test/resources/testdata
- Feature files in src/test/resources/features
